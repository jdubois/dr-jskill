# SKILL

## Introduction

In this document, we clarify the necessities for different types of projects. For **full stack projects**, having a PostgreSQL database is essential. Conversely, it's not required for other project categories. 

## Prerequisites

- For **full stack projects**:  
  - PostgreSQL database setup is required.  
  - Other prerequisites specific to the project type.  

- For other projects: 
  - No PostgreSQL database is necessary.  
  - Other prerequisites as applicable.  

## Best Practices

- When working on **full stack projects**:  
  - Ensure the PostgreSQL database is properly set up and accessible.  
  - Follow other best practices relevant to full stack development.  

- For different types of projects, focus on their individual best practices without the need for a PostgreSQL database.
